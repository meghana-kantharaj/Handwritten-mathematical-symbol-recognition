Simple handwritten math symbol dataset readme

This file contains the data set used to train the symblo classifier for this project. The dataset contains a total of 3500 handwritten symbols, 35 symbols each image. 

5*7*9=315 inputs for '-' sign(subtract)
5*7*10=350 inputs for '+' sign(add)
5*7*11=385 inputs for 'x' sign(multiply)
5*7*10=350 inputs for '/' sign(divide)
5*7*10=350 inputs for ')' sign(right parenthesis)
5*7*10=350 inputs for '(' sign(left parentesis)
5*7*10=350 inputs for '>' sign (greater than)
5*7*10=350 inputs for '<' sign (lesser than)
5*7*10=350 inputs for ']' sign (right square bracket)
5*7*10=350 inputs for '[' sign (left square bracket)

Size of the dataset  = 3.5MB

This data set was created for symbol recognition in simple math handwritten equations.
